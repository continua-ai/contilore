# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

We are committed to a welcoming and harassment-free community for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent.
- Focus feedback on ideas and code, not people.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Publishing private information without consent.
- Any conduct that makes participation unsafe.

## Reporting

Please report conduct issues to the maintainers through private channels.
